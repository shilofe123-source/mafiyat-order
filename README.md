# מאפיית השומרון — מערכת הזמנות מגשי אירוח

טופס הזמנה אונליין עם צ'אט AI מוטמע (Base44 bakeryChat + fallback מקומי).

## עיצוב ותכונות

- **עיצוב**: רקע משתנה עם תמונות מוצרים (slideshow כל 8 שניות) ושכבת שקיפות לקריאות
- **תהליך הזמנה**: בחירת מוצרים > אישור > שמירה בגיליון Google Sheets > שליחת PDF בוואטסאפ
- **כשרות**: מהדרין רבנות קרני שומרון
- **צ'אט AI**: ייעוץ אוטומטי ללקוחות — המלצות לפי סוג אירוע, עונה, כמות אורחים. הסוכן יכול גם למלא פרטי לקוח ולהוסיף מוצרים לעגלה
- **מצב עורך**: הגדרת הנחות, העלאת תמונות או PDF לחילוץ הזמנה אוטומטי (PIN: S199)

## אינטגרציות

- **Google Sheets** — שמירת הזמנות אוטומטית (תאריך, לקוח, טלפון, פריטים, סכום). במצב עורך — דריסת הזמנה קיימת לפי שם+טלפון. גיליון "סיכום" מתעדכן אוטומטית עם שם לקוח וסכום סופי (אחרי הנחה)
- **WhatsApp** — שליחת PDF הצעת מחיר ישירות ללקוח/לבעלים
- **Gemini 2.5 Flash** — חילוץ הזמנות מתמונות ו-PDF (דרך Apps Script)
- **Base44 AI** — צ'אט ייעוץ חכם (`bakeryChat` endpoint עם Claude), עם fallback מקומי אם ה-API לא זמין

## עדכון מוצרים ומחירים

ב-`index.html` יש שני מערכי `PRODUCTS`:

1. **מערך ראשי** (שורה ~149) — משמש את טופס ההזמנה:
   ```js
   { id: 1, name: "מגש כריכוני ביס", price: 180, qty_desc: "12 יח'", cat: "מגשי אירוח" }
   ```
   - `id` — מזהה ייחודי
   - `name` — שם המוצר
   - `price` — מחיר בשקלים
   - `qty_desc` — תיאור כמות למגש
   - `cat` — קטגוריה (מגשי אירוח / מאפים חמים / מנות חמות / פלטות / סלטים / פירות וקינוחים)

2. **מערך צ'אט** (שורה ~811) — משמש את ה-AI לייעוץ, כולל שדות `id`, `tags` ו-`category`

בעת עדכון מוצר/מחיר — יש לעדכן **בשני** המערכים. ה-`id` חייב להיות זהה בשניהם.

## ארכיטקטורה — גשר React ↔ Vanilla JS

הצ'אט (vanilla JS) מתקשר עם React דרך פונקציה גלובלית:
- `window.__addItemsFromChat(items)` — מוסיף מוצרים לעגלה (merge, לא replace)
- `fillForm(data)` — ממלא פרטי לקוח דרך DOM events + מוסיף מוצרים דרך הגשר

### מנגנון מילוי אוטומטי (AI → טופס)
1. כשה-checkbox "אפשר לסוכן למלא" מסומן, ה-system prompt מורה ל-AI להחזיר JSON עם `{"fill": {...}}`
2. ה-AI חייב לכלול `items` עם ה-`id` הנכון מרשימת המוצרים (כמות ברירת מחדל: 1)
3. תגובת ה-AI מנוקה מ-markdown wrapping לפני פרסור
4. שדות לקוח שלא ידועים ל-AI פשוט לא נכללים ב-JSON (לא נדרסים)

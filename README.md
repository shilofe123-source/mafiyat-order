# 🏪 מאפיית השומרון — טופס הזמנה ללקוחות

טופס הזמנה אונליין שנשלח ישירות לוואטסאפ של בעל העסק.

## 🚀 הוראות העלאה ל-GitHub Pages

### שלב 1 — צור חשבון GitHub (אם אין)
היכנס ל-[github.com](https://github.com) ופתח חשבון חינם.

### שלב 2 — צור Repository חדש
1. לחץ על **+** בפינה הימנית העליונה → **New repository**
2. שם: `mafiyat-order` (או כל שם אחר)
3. סמן **Public**
4. לחץ **Create repository**

### שלב 3 — העלה את הקבצים
1. בעמוד ה-repo החדש, לחץ **uploading an existing file**
2. גרור לתוך החלון את שני הקבצים:
   - `index.html`
   - `logo.jpeg`
3. לחץ **Commit changes**

### שלב 4 — הפעל GitHub Pages
1. לך ל-**Settings** (בתפריט העליון של ה-repo)
2. בצד שמאל לחץ על **Pages**
3. תחת **Source** בחר **Deploy from a branch**
4. תחת **Branch** בחר **main** ו- **/ (root)**
5. לחץ **Save**
6. תוך 1-2 דקות האתר יהיה זמין ב:
   `https://YOUR-USERNAME.github.io/mafiyat-order/`

### שלב 5 — עדכן מספר טלפון!
1. לחץ על `index.html` ב-GitHub
2. לחץ על העיפרון (Edit)
3. חפש `972XXXXXXXXX` ושנה למספר האמיתי של אורי
4. לחץ **Commit changes**

---

## 📱 איך הלקוח משתמש?

1. אורי שולח ללקוח קישור לאתר
2. הלקוח בוחר מוצרים, ממלא פרטים
3. לוחץ "שלח בוואטסאפ" → נפתח וואטסאפ עם הזמנה מפורמטת
4. או לוחץ "שמור PDF" → מוריד/מדפיס PDF לשליחה

## ✏️ איך לעדכן מחירים/מוצרים?

ב-`index.html` יש מערך `PRODUCTS` — פשוט לערוך שם:
- `name` — שם המוצר
- `price` — מחיר
- `qty_desc` — תיאור כמות
- `cat` — קטגוריה

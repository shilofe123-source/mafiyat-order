# מאפיית השומרון — מערכת הזמנות מגשי אירוח

טופס הזמנה אונליין עם צ'אט AI מוטמע (Base44 bakeryChat + fallback מקומי).

## עיצוב ותכונות

- **עיצוב**: רקע כחול-שמנת (blue-cream gradient)
- **תהליך הזמנה**: בחירת מוצרים > אישור > שמירה בגיליון Google Sheets > שליחת PDF בוואטסאפ
- **כשרות**: מהדרין רבנות קרני שומרון
- **צ'אט AI**: ייעוץ אוטומטי ללקוחות — המלצות לפי סוג אירוע, עונה, כמות אורחים
- **מצב עורך**: הגדרת הנחות, העלאת תמונות (PIN: 2001)

## קבצים

| קובץ | תיאור |

| `index.html` | האפליקציה המלאה — React + HTM, אפס תלויות שרת |

## אינטגרציות

- **Google Sheets** — שמירת הזמנות אוטומטית (תאריך, לקוח, טלפון, פריטים, סכום)
- **WhatsApp** — שליחת PDF הצעת מחיר ישירות ללקוח/לבעלים
- **Base44 AI** — צ'אט ייעוץ חכם (`bakeryChat` endpoint), עם fallback מקומי אם ה-API לא זמין

## עדכון מוצרים ומחירים

ב-`index.html` יש שני מערכי `PRODUCTS`:

1. **מערך ראשי** (שורה ~149) — משמש את טופס ההזמנה:
   ```js
   { id: 1, name: "מגש כריכוני ביס", price: 180, qty_desc: "12 יח'", cat: "מגשי אירוח" }
   ```
   - `id` — מזהה ייחודי
   - `name` — שם המוצר
   - `price` — מחיר בשקלים
   - `qty_desc` — תיאור כמות למגש
   - `cat` — קטגוריה (מגשי אירוח / מאפים חמים / מנות חמות / פלטות / סלטים / פירות וקינוחים)

2. **מערך צ'אט** (שורה ~771) — משמש את ה-AI לייעוץ, כולל שדות `tags` ו-`category`

בעת עדכון מוצר/מחיר — יש לעדכן **בשני** המערכים.
